What AEM Edge Delivery gives you

Before a single line of custom code, every EDS project ships with a set of capabilities that make it faster and easier to run than any traditional CMS.

Edge Performance

Code splits per block, loads at the edge, and ships zero render-blocking resources

Document Authoring

Authors write in da.live — a familiar document editor..

Global CDN

Content is cached at Adobe's edge network and invalidated automatically on publish.

Block Architecture

Every section of a page loads its own isolated CSS and JS. Only what's visible is ever downloaded.

Image Optimisation

Every image is automatically resized, converted to WebP, and lazy-loaded.

Semantic HTML & SEO

Markup is generated from document structure — clean headings, proper landmarks.

What customisation adds on top

The platform gives you the rails. Custom blocks let you extend it into something distinctly yours — without sacrificing the performance baseline.

Standard AEM

Out of the box

  • Fast edge delivery with PageSpeed 100
  • Document-based authoring in da.live
  • Automatic image optimisation
  • Generic header and footer fragments
  • Basic responsive grid layout
  • Preview → Publish content workflow
With Custom Blocks

What we built on top

  • Live search by title, tag and date
  • Tag filter chips — auto-built from content
  • Paginated card grid (8 cards per page)
  • Staggered scroll entrance animations
  • Brand-consistent design token system
  • Microsoft-style Segoe UI typography
